The correct answer is: It is the same as the secondary current.

Why?

In a step-up autotransformer, the series winding is connected in series with the load. Because it is positioned between the tap point and the output terminal, all the current flowing to the load (the secondary current) must pass through it.

Key Current Relationships

To keep things clear, here is how the currents distribute in a step-up configuration:

Current Type Relationship
Series Winding Current (IserI_{ser}) Equal to the Secondary Current (I2I_2).
Common Winding Current (IcomI_{com}) The difference between the primary and secondary currents: I1−I2I_1 - I_2.
Primary Current (I1I_1) The sum of the common winding current and the series winding current.

Quick Rule of Thumb

  • Step-Up: Series winding current = Secondary current.
  • Step-Down: Series winding current = Primary current.
