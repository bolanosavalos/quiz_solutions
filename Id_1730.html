Here is the correct calculation using the Locked Rotor Current (LRC) method to find the maximum time-delay fuse ceiling under the Canadian Electrical Code (CEC):

1. Determine Full Load Current (FLC)

The FLC is given as 45A.

2. Calculate Locked Rotor Current (LRC)

Under CEC Section 28 rules for hermetic refrigerant motor-compressors, when the LRC isn't explicitly specified on the nameplate, it is considered to be 6 times the FLC.

  • 45A x 6 = 270A

3. Apply the Time-Delay Fuse Limit

The code mandates that a time-delay fuse cannot exceed 50% of the motor's Locked Rotor Current (if you were using a non-time-delay fuse, this multiplier would be 0.65).

  • 270A x 0.5 = 135A

4. Select the Standard Fuse Size

Using CEC Table 13 for standard overcurrent device ratings, the available sizes in this range are 110A, 125A, and 150A. Because the rule states the fuse rating shall not exceed 135A, you must drop down to the next standard size.

  • Maximum Standard Fuse = 125A
