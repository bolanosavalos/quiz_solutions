The statement is False because the Canadian Electrical Code (CEC) Rule 18-158 requires that flexible cords used in a Zone 2 hazardous location must be of the extra-hard usage type.

Explanation

  • Rule 18-158 (1) in the CEC permits flexible cords in Zone 2 locations only where flexibility is required, such as for portable lamps or utilization equipment.
  • The rule specifies that such flexible cords must be of the extra-hard usage type and meet other conditions (like containing a bonding conductor and proper termination).
  • The flexible cord type SJOW is typically rated for junior hard service (the 'J' in the designation, which also corresponds to a 300V rating), not the required extra-hard usage. Extra-hard usage cords are typically designated as SOOW, SEOOW, or similar, and are rated for 600V (no 'J').

Therefore, a Type SJOW flexible cord is not permitted for this application in a Zone 2 location because it does not meet the minimum durability requirement of being an extra-hard usage type cord.
