Based on the Canadian Electrical Code (CEC) Rule 28-708(1) and the scenario where the existing overcurrent protection blows during the motor's starting phase, the replacement fuse value needs to be increased.

Rule 28-708(1) is generally interpreted to allow for a maximum size of the overcurrent device, for a hermetic motor compressor with a locked-rotor current (LRC) marked on the equipment, to be:

  • 50% of the Locked-Rotor Current (LRC) for a fuse (or non-time-delay circuit breaker).
