Here is the step-by-step calculation based on the Canadian Electrical Code (CE Code) for a cold-start/hermetic freezer motor:

1. Find the Full Load Current (FLC)

According to Table 44 (Full load current, single-phase AC motors), the FLC for a 5 HP motor at 208V is 30.8 A.

2. Calculate the Locked Rotor Current (LRC)

For hermetic refrigerant motor-compressors where the LRC is not explicitly marked on the nameplate, Rule 28-704 dictates that it is considered to be 6 times the rated load current.

  • 30.8 A x 6 = 184.8 A

3. Apply the Maximum "Cold Start" Limit

To allow the compressor to start under difficult conditions without nuisance tripping, Rule 28-708 allows the time-delay fuse rating to be increased, but it shall in no case exceed 50% of the motor's Locked Rotor Current.

  • 184.8 A x 0.5 = 92.4 A

4. Select the Maximum Fuse Rating

Because the Code strictly states the overcurrent device cannot exceed 92.4 A, you must select the largest standard fuse size that stays under this absolute limit.

  • Looking at your options Table 13 (95A, 90A, 100A, 80A), a 95A fuse would violate the Code by exceeding the 92.4 A maximum.
  • The correct maximum allowable rating is 90A.
