CEC Voltage Drop Calculation

The CEC 2024 uses the following formula to calculate conductor size when voltage drop is the limiting factor:

K=Vdrop×1000f×I×LK = \frac{V_{\text{drop}} \times 1000}{f \times I \times L}

Where:

  • KK = Conductor resistance (Ω/km\Omega / \text{km}) from Table D3.
  • VdropV_{\text{drop}} = Maximum allowable voltage drop (Volts).
  • ff = System factor (2 for single-phase 120 V120 \text{ V} circuit).
  • II = Current (A\text{A}).
  • LL = One-way length (m\text{m}).
