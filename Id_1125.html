The maximum size of a time-delay fuse for a 3-phase, 20 HP, 600V squirrel cage induction motor, limited to the next standard size equal to or less than 175% of FLA, is 35 A based on the Canadian Electrical Code (CEC).

CEC Rule Breakdown (Rule 28-200)

The sizing is determined by the Full Load Amperes (FLA) of the motor and the maximum percentage multiplier specified in CEC Rule 28-200 and Table 29.

1. Full Load Amperes (FLA)

  • Motor: 3-phase, 20 HP, 600 V
  • Reference: CEC Table 44
  • FLA: 22 A

2. Maximum Multiplier (Standard Starting Condition)

For a squirrel-cage induction motor protected by a time-delay fuse, the maximum rating for short-circuit and ground-fault protection is 175% of the FLA.

  • Reference: CEC Table 29
  • Multiplier: 175% or 1.75

3. Calculated Maximum Current

Calculated Max Current=FLA×Multiplier\text{Calculated Max Current} = \text{FLA} \times \text{Multiplier} Calculated Max Current=22 A×1.75=38.5 A\text{Calculated Max Current} = 22 \text{ A} \times 1.75 = \mathbf{38.5 \text{ A}}

4. Selecting the Standard Fuse Size

Standard fuse sizes are listed in CEC Table 13. Since 38.5 A is not a standard size, we must select a standard size from Table 13.
