üí° CEC Rule 8-202(1)(a)(i) and (ii)

The basic load for a dwelling unit in an apartment or similar building is calculated using the living area, as per the Canadian Electrical Code (CEC) Rule 8-202(1)(a)(i) and (ii) (in conjunction with Rule 8-110, which defines living area):

  • Rule 8-202(1)(a)(i): A basic load of 5000 W for the first 90 m2\text{m}^2 or less of living area.
  • Rule 8-202(1)(a)(ii): An additional 1000 W for each 90 m2\text{m}^2 or portion thereof in excess of the first 90m290 \text{m}^2.
