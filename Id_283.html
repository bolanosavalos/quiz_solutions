The maximum allowable ampacity for the SOO flexible cord with six #8 copper current-carrying conductors is 28 A.

⚡ Ampacity Determination: CEC Rule 4-012

This calculation involves two main steps as required by the Canadian Electrical Code (CEC) Rule 4-012: finding the base ampacity and then applying the correction factor for the high number of conductors.

1. Identify the Base Ampacity (Table 12)

The base ampacity is taken from Table 12 under the column designated for 3 current-carrying conductors (CCC), as this is the standard reference point for derating:

  • Conductor Size: #8 AWG Copper
  • Cord Type: SOO Flexible Cord
  • Value in the 3 CCC Column: 35 A

Base Ampacity=35 A\text{Base Ampacity} = \mathbf{35\text{ A}}

2. Apply the Correction Factor (Rule 4-012)

The CEC requires a derating factor (correction factor) to be applied because the cord has more than three current-carrying conductors:

  • Total CCC: 6
  • Rule 4-012(c) states that for 4 to 6 insulated conductors, the maximum current is limited to 80% (0.80) of the value in Table 12.

Correction Factor=0.80\text{Correction Factor} = \mathbf{0.80}

3. Final Calculation

Multiply the base ampacity by the correction factor:

Maximum Ampacity=Base Ampacity×Correction Factor\text{Maximum Ampacity} = \text{Base Ampacity} \times \text{Correction Factor}

Maximum Ampacity=35 A×0.80=28 A\text{Maximum Ampacity} = 35\text{ A} \times 0.80 = \mathbf{28\text{ A}}

The correct maximum allowable ampacity for this cord is 28 A.
