The type designation 2B from the Canadian Electrical Code, Part I (18th edition, 1998) Table B62-1 identifies a trace heater set intended for:

❄️ Snow Melting and Embedded Floor Applications

The full description for designation 2B is:

  • Usage: Snow melting and floor embedded in concrete (wet location).

This designation specifically confirms the heater's suitability for being completely embedded in the concrete or similar non-combustible material of an outdoor slab for the purpose of snow and ice removal.

The table indicates that type designations starting with 'W' (like 2B falls under the umbrella of 'Previous type designations and applications' for the 'W' usage marking) are generally used for:

  • Usage Marking: W (With wet rating)
  • Wet location applications: Yes
  • Wet location under pressure: No

While the current edition of the CEC (2021) doesn't use these specific "2B" type designations, the underlying requirement for snow melting cable sets remains the same: they must be robust, wet-rated, and certified to the appropriate CSA standard (like CAN/CSA C22.2 No. 130), and installed according to the rules in CEC Section 62.
