For extra-low-voltage wiring (such as HVAC control wiring) that must be run through a plenum or air-handling space, the Canadian Electrical Code (CEC) requires the cable to be rated for installation in a plenum space - meaning it must have appropriate flame-spread and smoke characteristics suitable for that environment. CEC references the National Building Code of Canada (NBC) flame spread requirements (FT ratings) for cables in plenum spaces and requires a plenum-rated jacket (e.g., FT-6) for exposed cables not in conduit.

Among the answer choices provided:

  • a. Nylon cable - not a recognized required type for plenum wiring.
  • b. Flat conductor cable - a cable construction, not specifically plenum-rated on its own.
  • c. Non-metallic sheathed cable (NM) - this is not permitted in plenum spaces unless it is specifically rated for plenum use, which standard NM is not.
  • **d. Control and instrumentation cable - this is the correct category of cable for HVAC extra-low-voltage control circuits; provided it is plenum-rated (e.g., FT-6 or equivalent jacket suitable for plenum use).

✅ Correct Answer: d. Control and instrumentation cable (with a plenum-rated jacket suitable for HVAC plenum spaces per CEC/NBC fire-safety requirements).

In practice, you’d select control/instrumentation cable that is specifically listed for plenum use (with appropriate flame/spread rating such as FT-6) when routing through ventilation plenums in accordance with code requirements.
