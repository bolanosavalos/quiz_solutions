To find the current on the neutral conductor, we first need to determine the current flowing through each individual phase (or "leg") of the 120/240 V system.

In a standard North American residential split-phase system, each phase leg provides 120 V relative to the neutral wire.

Step 1: Calculate the current for each phase

We use the power formula P=V×IP = V \times I, rearranged to solve for current: I=PVI = \frac{P}{V}.

  • Phase A Current (IAI_A): IA=1200 W120 V=10 AI_A = \frac{1200\text{ W}}{120\text{ V}} = 10\text{ A}
  • Phase B Current (IBI_B): IB=7200 W120 V=60 AI_B = \frac{7200\text{ W}}{120\text{ V}} = 60\text{ A}

Step 2: Calculate the neutral current

In a three-wire, single-phase system, the neutral wire carries only the unbalanced portion of the load. Because the two hot legs are 180° out of phase, their currents oppose each other on the neutral wire.

The formula for neutral current (INI_N) is: IN=∣IA−IB∣I_N = |I_A - I_B|

  • Neutral Current Calculation: IN=∣10 A−60 A∣=50 AI_N = |10\text{ A} - 60\text{ A}| = 50\text{ A}

Conclusion

The current on the neutral conductor is 50 A.
