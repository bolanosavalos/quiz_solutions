The slip RPM (Nslip\mathbf{N_{slip}}) of the motor at full load is 18 RPM.

Calculation Steps\text{Calculation Steps}

The slip RPM for a motor is calculated using the formula: Nslip=s×NsN_{slip} = s \times N_s where:

  • s\mathbf{s} is the per unit slip.
  • Ns\mathbf{N_s} is the synchronous speed in revolutions per minute (RPM).

1. Calculate the Synchronous Speed (Ns)1.\ \text{Calculate the Synchronous Speed} \ (N_s)

The synchronous speed is determined by the number of poles (P\mathbf{P}) and the supply frequency (f\mathbf{f}): Ns=120×fPN_s = \frac{120 \times f}{P}

Given:

  • f=60 Hz\mathbf{f = 60 \ \text{Hz}}
  • P=6 poles\mathbf{P = 6 \ \text{poles}}

Ns=120×60 Hz6 poles=72006=1200 RPMN_s = \frac{120 \times 60 \ \text{Hz}}{6 \ \text{poles}} = \frac{7200}{6} = 1200 \ \text{RPM}

2. Calculate the Slip RPM (Nslip)2.\ \text{Calculate the Slip RPM} \ (N_{slip})

Given:

  • s=1.5%=1.5100=0.015 (per unit)\mathbf{s} = 1.5 \% = \frac{1.5}{100} = 0.015 \ \text{(per unit)}
  • Ns=1200 RPM\mathbf{N_s = 1200 \ \text{RPM}}

Nslip=s×NsN_{slip} = s \times N_s Nslip=0.015×1200 RPM=18 RPMN_{slip} = 0.015 \times 1200 \ \text{RPM} = 18 \ \text{RPM}
