In the Canadian Electrical Code (CEC) 2024, when you’re running Cat5e communications cable through a plenum space (e.g., above a drop ceiling or in a duct where air circulates), the cable must be a plenum-rated communications cable. In code terms, this means a CMP-rated cable. That rating correlates with FT-6 flame/smoke performance under Canadian cabling fire tests which is required for exposed cables in plenum spaces.

How to find it in CEC 2024 (using Table 19)

  • Table 19 in the CEC is the primary table for identifying appropriate cable types and their designated applications. It lists types of insulated conductors and cables and, where they can be used.
  • For communications cables that will be outside of a raceway in a plenum air-handling space, the code requires a cable that meets the more stringent flame-spread and smoke-production tests for plenum use. In practice, this is shown by the CMP marking (Communications Plenum).
  • Table 19 (and related rules with Appendix B notes) tie the CEC designation to the flame test categories (FT-6) used in the National Building Code of Canada for plenums.

Answer to your multiple-choice

C. CMP — Communications Plenum rated cable is the correct choice for Cat5e wire run from Joint to Patch in a plenum space in accordance with CEC Table 19 and the applicable plenum fire-safety requirements. 

Explanation of other options (why they’re wrong):

  • A. None of these — Not correct, because there is a required rating.
  • B. FCC — “FCC” isn’t a cable type designation in CEC; it’s the U.S. Federal Communications Commission, unrelated here.
  • D. SOP — “SOP” isn’t a standard CEC cable rating term.

Summary: Use CMP (plenum)-rated Category-rated communications cable for plenum runs per CEC Table 19 and fire/smoke code requirements.
