Based on the Canadian Electrical Code (CEC) Rule 42-008, the maximum rating for the overcurrent protection device for this transformer arc welder is 96 Amps96\text{ Amps}.

CEC Calculation

CEC Rule 42-008(1) states that overcurrent protection for a transformer arc welder shall be rated or set at not more than 200%200\% of the rated primary current of the welder.

Maximum Overcurrent Protection=Rated Primary Current×200%\text{Maximum Overcurrent Protection} = \text{Rated Primary Current} \times 200\% Maximum Overcurrent Protection=48 A×2.0=96 Amps\text{Maximum Overcurrent Protection} = 48\text{ A} \times 2.0 = \mathbf{96\text{ Amps}}

Note on Standard Ratings

In an actual installation, since 96 Amps96\text{ Amps} is not a standard overcurrent device rating, an electrician would typically select the nearest standard rating not exceeding 96 Amps96\text{ Amps}, which is 90 Amps\mathbf{90\text{ Amps}}.
