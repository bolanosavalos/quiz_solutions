The Calculation

To find the maximum number of conductors, divide the total box volume by the volume required for a single wire:

Maximum Conductors=163 mL28.7 mL/conductor≈5.68\text{Maximum Conductors} = \frac{163\text{ mL}}{28.7\text{ mL/conductor}} \approx 5.68

Final Determination

The Code does not allow for rounding up, as that would result in a "box fill" violation. Therefore, the absolute maximum number of #12 AWG conductors allowed is 5.
