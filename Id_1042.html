For a 22 HP, 230 V, 3-phase induction motor, the correct FLC from CEC Table 44 (or equivalent standard tables) must be used. Since 22 HP is not a standard listing in many simplified tables, the value is interpolated or taken from a more complete table.

The correct FLC for a 20 HP, 230 V, 3-phase motor is 54 A, and the FLC for a 25 HP, 230 V, 3-phase motor is 68 A (as you noted). Interpolating for 22 HP is complex, so we typically use the next standard size up (25 HP) when the exact HP is not listed, or we use the FLC for the nearest standard HP that is at least the required HP. However, the exact FLC for 22 HP is sometimes cited as 59 A in full tables.

Let's proceed with the FLC for the next higher standard HP (25 HP) as a conservative approach, which gives 68 A, since the motor is slightly above 20 HP.

3. Select Standard Fuse Size (CEC Table 13)

Standard fuse sizes near 119 A are 110 A, 125 A, etc.

  • The required maximum setting based on the initial calculation is 119 A.
  • CEC Rule 28-200(4)(a) permits using the next standard size of fuse from Table 13 if the calculated value (119 A) does not match a standard size.
  • Next Standard Size Above 119 A: 125

4. Determine Absolute Maximum Fuse Rating

If the motor fails to start with the 125 A fuse, CEC Rule 28-200(4)(b) allows increasing the rating, but not to exceed 225% of the FLC for a time-delay fuse when IFL≤100 AI_{\text{FL}} \le 100\text{ A}.

  • Absolute Maximum Calculated: Iabsolute max=68 A×2.25I_{\text{absolute max}} = 68\text{ A} \times 2.25 Iabsolute max=153 AI_{\text{absolute max}} = \mathbf{153\text{ A}}

  • Since the calculated absolute maximum (153 A\text{153 A}) is not a standard size, you would use the next lower standard size that does not exceed the calculated value, unless the motor cannot start. Standard fuse sizes are 150 A and 175 A.

  • Absolute Maximum Standard Size (Not Exceeded): 150 A

Therefore, the absolute maximum time-delay fuse allowed for this motor is 150 A.
