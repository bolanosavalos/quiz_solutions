Correct answer: The current carried by the neutral conductor will be zero amps.

Explanation

In a three‑phase four‑wire delta system, the neutral (or center tap) exists only if one transformer in the delta is center‑tapped—typically the high‑leg delta configuration used for 120/240 V + 240 V three‑phase.

However:

  • A delta-connected load does not use the neutral.
  • All three phase conductors form closed delta loops.
  • No portion of the delta load is connected to the center tap / neutral conductor.

Because the neutral has no return path connected through the load, no current can flow in the neutral, even if the delta load is unbalanced.

📌 Unbalanced currents circulate inside the delta loop, not through the neutral.

Therefore:

The neutral current will be zero amps.
