To find the new amperage when the battery voltage drops, we can use Ohm's Law, which states:

V=I×RV = I \times R

Where:

  • VV is voltage (volts)
  • II is current (amperes)
  • RR is resistance (ohms)

1. Calculate the Load Resistance

First, we find the resistance of the load using the initial conditions (V=12 VV = 12\text{ V}, I=10 AI = 10\text{ A}): R=VI=12 V10 A=1.2 ΩR = \frac{V}{I} = \frac{12\text{ V}}{10\text{ A}} = 1.2\ \Omega

2. Calculate the New Voltage

The problem states the battery voltage dropped by 10%10\%. Vnew=12 V−(12 V×0.10)=12 V−1.2 V=10.8 VV_{\text{new}} = 12\text{ V} - (12\text{ V} \times 0.10) = 12\text{ V} - 1.2\text{ V} = 10.8\text{ V}

3. Calculate the New Amperage

Since the load resistance remains the same (1.2 Ω1.2\ \Omega), we calculate the new current: Inew=VnewR=10.8 V1.2 Ω=9 AI_{\text{new}} = \frac{V_{\text{new}}}{R} = \frac{10.8\text{ V}}{1.2\ \Omega} = 9\text{ A}

Answer: The amperage would be 9 amperes9\text{ amperes}.
