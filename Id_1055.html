To reverse the rotation of a capacitor-start motor, you must reverse the connections of the running coil leads (the main winding) relative to the starting winding.

  • How it works: Reversing the connections to the running coil changes the direction of the magnetic field it creates. This, in turn, changes the phase relationship between the magnetic fields of the running winding and the starting winding, causing the motor to start and run in the opposite direction.
  • Why not the others?
    • a) running and starting coil leads: Reversing both sets of leads simultaneously would maintain the same relative phase relationship between the two windings, and the motor's direction would not change.
    • b) any of these: This is incorrect as reversing both windings or just the capacitor connection won't work.
    • d) connection to the capacitor: Reversing the capacitor connection does not change the motor's rotation direction. The capacitor's role is to create a phase shift (approximately 90âˆ˜90^\circ) for the starting winding's current relative to the running winding's current, which is necessary to generate the rotating magnetic field to start the motor. Reversing its connection doesn't alter this phase shift or the relative direction of the magnetic fields.
