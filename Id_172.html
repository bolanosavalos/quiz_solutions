Rule 64‑210(1) – Interconnection of PV Modules

  • It specifies that flexible cords used to interconnect PV modules must be rated for extra‑hard usage (often outdoor‑rated and sunlight‑resistant).
  • The CEC Table 11 cross‑references cord types suitable for this, and the standard type listed is:

✅ STOO – Thermoplastic‑insulated, extra‑hard usage flexible cord

Why STOO?

Cable Type Usage per CEC Table 11 / Rule 64‑210(1)
STOO ✔ Permitted for PV module interconnection (extra‑hard usage, sunlight-resistant)
SJOO ✖ Hard usage, but not rated for outdoor UV exposure per CEC for PV modules
SJOW ✖ Hard usage, but less commonly listed in CEC Table 11 for PV arrays
SEWF‑2 ✖ Used for fixed wiring (e.g., branch circuits), not flexible cords

So the correct type of cable for interconnecting PV modules per Rule 64‑210(1) and Table 11 is STOO.
